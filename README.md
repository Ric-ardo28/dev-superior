# 📚 DevSuperior - Curso de Java com Spring Boot

Este repositório contém as atividades, desafios e projetos desenvolvidos ao longo do curso **Java com Spring Boot** ministrado pelo professor **Nélio Alves** na plataforma [DevSuperior](https://devsuperior.club/courses/catalogue).

## 📌 Sobre o curso

O curso tem como objetivo ensinar os fundamentos e práticas do desenvolvimento backend com Java, utilizando o ecossistema Spring, incluindo:

- Spring Boot
- Inversão de Controle (IoC) e Injeção de Dependência (DI)
- Spring Data JPA
- Acesso a banco de dados com Hibernate
- Criação de APIs RESTful
- Testes e boas práticas
